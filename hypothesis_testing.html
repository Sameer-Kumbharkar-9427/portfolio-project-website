Project Overview

In this project, I analyzed a website’s Light and Dark themes using user engagement metrics such as Click Through Rate (CTR), Conversion Rate, Bounce Rate, and Scroll Depth.

After exploratory data analysis, I conducted hypothesis testing to determine statistically significant differences in user behavior across the two themes.

The analysis revealed that while most metrics showed no significant variation, the Dark Theme demonstrated a statistically higher Click Through Rate, suggesting better initial user engagement.

Data Description ▼

The dataset includes 1,000 records with the following columns:

  • Theme: Indicates whether the user saw the Light or Dark theme.
  • Click Through Rate: The percentage of users who clicked on a link.
  • Conversion Rate: The percentage of users who completed a desired action.
  • Bounce Rate: The percentage of users who left the site after viewing only one page.
  • Scroll Depth: How far users scrolled down the page.
  • Age: Age of the users
  • Location: Geographical location of the user.
  • Session Duration: Time spent on the website during a session.
  • Purchases: Whether the user made a purchase.
  • Added_to_Cart: Whether the user added items to the cart.
