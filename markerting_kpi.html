Project Overview

In this project, I created a MySQL database and imported a dataset containing one month's digital marketing campaign data. After importing, I standardized and cleaned the dataset to ensure consistency and accuracy.

I calculated key marketing KPIs such as ROMI, CTR, CAC, CPC, CPL, and Gross Profit, and created a database view for structured reporting.

The view was imported into Power BI, where I developed custom measures to standardize financial values into Indian denominations. I built a three-page dashboard:

Page 1: KPIs based on Acquisition, Conversion, and Revenue
Page 2: Campaign-wise insights including Average CAC, ROMI, AOV, and CAC vs CPC
Page 3: Category-wise insights with the same set of KPIs

This project enhanced my skills in SQL data processing, KPI calculation, Power BI visualization, and digital marketing analytics.

    -- Standardizing data by making column lower case

    update marketing
    set campaign_name = lower(campaign_name);

    -- Calculating KPI's
    -- ROMI (Return On Marketing Cost)  
    
    select (sum(revenue) - sum(mark_spent))/sum(mark_spent)*100 as ROMI
    from marketing;
   
    

    -- CTR (Click Through Rate)

    select  campaign_name, sum(clicks)/sum(impressions)*100 as CTR
    from marketing
    group by campaign_name, category
    order by CTR desc;
    

    -- Conversion(leads/clicks and orders/leads)

    select campaign_name, sum(leads)/sum(clicks)*100 as conversion_1
    from marketing
    group by campaign_name
    order by conversion_1 desc;
        
    select campaign_name, sum(orders)/sum(leads)*100 as conversion_2
    from marketing
    group by campaign_name
    order by conversion_2 desc;
    

    -- AOV (Average Order Value)

    select campaign_name, round(sum(revenue)/sum(orders), 2) as AOV
    from marketing
    group by campaign_name
    order by AOV desc;    
   
    

    -- CPC (Cost Per Click)

    select campaign_name, round(sum(mark_spent)/sum(clicks), 2) as CPC
    from marketing
    group by campaign_name
    order by CPC desc;
   
    

    -- CPL (Cost Per Lead)

    select campaign_name, round(sum(mark_spent)/sum(leads), 2) as CPL
    from marketing
    group by campaign_name
    order by CPL desc;
   
    

    -- CAC(Customer Acquisition Cost)

    select campaign_name, round(sum(mark_spent)/sum(orders), 2) as CAC
    from marketing
    group by campaign_name
    order by CAC desc;
   
    

    -- Gross Profit

    select round(sum(revenue) - sum(mark_spent), 2) as gross_profit
    from marketing;
   
    

    -- Create View    

    create view campaign_kpi as
    select
	    campaign_name,
        category,
        (sum(revenue) - sum(mark_spent))/sum(mark_spent)*100 as ROMI,
        sum(clicks)/sum(impressions)*100 as CTR,
        sum(leads)/sum(clicks)*100 as conversion_1,
        sum(orders)/sum(leads)*100 as conversion_2,
        round(sum(revenue)/sum(orders), 2) as AOV,
        round(sum(mark_spent)/sum(clicks), 2) as CPC,
        round(sum(mark_spent)/sum(leads), 2) as CPL,
        round(sum(mark_spent)/sum(orders), 2) as CAC
    from marketing
    group by campaign_name, category;

Actionable Insights ▼
  • The campaign achieved 157.8 Cr impressions and 30 Lakh clicks, resulting in a CTR of 0.19%, suggesting decent ad engagement but there's room for optimization.

  • Despite strong traffic, the click-to-lead conversion rate is only 0.22%, indicating the need to improve landing page effectiveness and lead capture processes.

  • The Average Customer Acquisition Cost (₹4370) is almost equal to the Average Order Value (₹4730), highlighting razor-thin margins and the need to either lower CAC or increase AOV.

  • YouTube Blogger campaigns delivered the highest performance with around 200% ROMI and highest revenue, indicating these campaigns should be scaled aggressively.
    Facebook lal campaigns showed high CAC and low ROMI,suggesting immediate optimization or pausing ofthese campaigns to prevent budget waste.

  • Social and Search categories generated higher volumes but with significantly lower ROMI, recommending a careful cost-control strategy while using them for volume generation.
